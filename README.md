# 🎨 Webcam Edge Detection — TouchDesigner

> A real-time interactive visual effect built in TouchDesigner — detects edges from a live webcam feed and applies rainbow colour cycling using sine wave expressions.


## ✨ What it does

- Captures live webcam feed
- Applies edge detection to create a glowing outline effect
- Colours automatically cycle through the rainbow in real time using `math.sin()` expressions
- Runs at 60 FPS

## 🛠 Built with

- **TouchDesigner 2025** — node-based visual programming
- **Nodes used:** Video Device In, Level, Edge, HSV Adjust, Out
- **Expressions:** Python `math.sin()` for colour animation

## 🚀 How to open

1. Download **TouchDesigner** free at [derivative.ca](https://derivative.ca)
2. Clone this repo
3. Open `NewProject.2.toe` in TouchDesigner
4. Allow webcam access when prompted
